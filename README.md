This repository contains our system for **SemEval-2026 Task 9, Subtask 1**, 
which focuses on **binary polarization detection** in multilingual social media text.

## Task Description
Given a social media post, the goal is to determine whether it contains
**attitude polarization** (1) or not (0).

## Languages
We experiment with Arabic language that released by the organizers.

## Model
- Transformer-based architecture (AraBERT)
- Binary classification head
- Macro F1-score used for evaluation

## Data
Data is provided by the SemEval-2026 organizers.
Please download it from the official website:
https://polar-semeval.github.io/
